## What is GeminiAnswerBot?

GeminiAnswerBot is a Chrome extension that uses Google's Gemini AI to help answer quiz questions. It automatically detects questions on web pages, analyzes them, and provides AI-generated answers with explanations.

## Getting Started

1. Install from Chrome Web Store (or load unpacked)
2. Get a [Gemini API key](https://aistudio.google.com/apikey)
3. Open extension settings and enter your API key
4. Navigate to any quiz page and click the extension icon
